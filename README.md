# 🚀 NEXUS AI SUITE

Complete AI-powered learning platform for Indian students. 100% free and open-source.

## ✨ Features

### 🎓 For Students
- **Python IDE** with AI Copilot (auto-save)
- **RoboBuilder** - 3D robot designer with price breakdown
- **Smart Canvas** - AI-predicted whiteboard
- **AI Chat** - 150+ models (GPT-4o, Claude, Gemini, etc.)
- **Study Helper** - CBSE/Board exam solver (Hindi + English)
- **Board of Advisors** - Connect with mentors

### ⚡ Premium Features (₹33/month)
- GPT-4o, Claude 3.5 Sonnet access
- Team Collaboration rooms
- Unlimited storage
- 24/7 support

### 👑 Admin Features
- Ban/Suspend users
- Upload advertisements
- Approve premium requests
- View analytics

## 🛠️ Tech Stack
- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: Supabase (PostgreSQL)
- **AI**: Google Gemini, OpenRouter (GPT-4, Claude)
- **Code Execution**: Piston API
- **Storage**: Supabase Storage
- **Deployment**: Vercel
